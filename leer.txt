esta es la version 4 donde veremos unos comandos de git :

1. **git init**: Inicializa un nuevo repositorio Git en tu directorio actual.
2. **git clone [url]**: Clona (descarga) un repositorio que existe en la URL especificada.
3. **git add [archivo]**: Agrega un archivo al área de preparación para el próximo commit.
4. **git commit -m "[mensaje]"**: Guarda tus cambios en el repositorio con un mensaje descriptivo.
5. **git status**: Muestra el estado del repositorio Git actual, incluyendo cambios sin commit.
6. **git pull**: Actualiza tu repositorio local con la versión más reciente del repositorio remoto.
7. **git push**: Sube tus commits al repositorio remoto.
8. **git branch**: Muestra las ramas del repositorio y te permite crear o eliminar ramas.
9. **git checkout [nombre_rama]**: Cambia a la rama especificada.
10. **git merge [nombre_rama]**: Fusiona la rama especificada con la rama actual.
